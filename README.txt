# Victor Shulist — Resume Website

This folder contains a lightweight, multi‑page static website built from the provided resume.

## Files
- index.html — Home & profile
- experience.html — Detailed professional experience
- skills.html — Areas of expertise
- certifications.html — Certifications, memberships, education, and badges
- contact.html — Contact information
- style.css — Styles (dark theme, responsive)
- script.js — Small enhancement for nav state
- assets/Victor-Shulist-Resume.txt — Plain text export placeholder

## How to use
Open `index.html` in a browser, or host the folder on any static host (GitHub Pages, Netlify, S3).
